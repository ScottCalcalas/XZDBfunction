
# DBfunction

Modularized R package with:
- roxygen2 documentation
- GitHub Actions (R CMD check)
- pkgdown website support

## Installation

```r
remotes::install_github("scottcalcalas/XZDBfunction")
```

## Build documentation

```r
devtools::document()
```

## Build pkgdown site

```r
pkgdown::build_site()
```
