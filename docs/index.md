
# XZDBfunction — Genomic DB Browser

**XZDBfunction** is an integrated R package supporting:

- ✔ Dataset indexing  
- ✔ Gene ID & Gene Symbol normalization  
- ✔ Family-based gene symbol matching  
- ✔ UniProt and MINT REST API comparison  
- ✔ Shiny-based interactive multi-omics browser  

## Install

```r
remotes::install_github("ScottCalcalas/XZDBfunction")
```

## Launch the Shiny App

```r
XZDBfunction::XZDB.Run()
```
